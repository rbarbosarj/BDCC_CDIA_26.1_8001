# 🌐 BDCC – Big Data & Cloud Computing

## 🎓 Informações Acadêmicas
*   **Curso:** Engenharia de Computação
*   **Período:** 2026.1
*   **Descrição:** Repositório central contendo as Atividades Curriculares (ACs) desenvolvidas ao longo da disciplina de Big Data & Cloud Computing.

---

## 📂 Estrutura do Repositório

O repositório está subdividido em pastas individuais para cada atividade do semestre:

*   **`AC1/`** – Computação em Nuvem
*   **`AC2/`** – Modelos de Serviço em Nuvem
*   **`AC3/`** – Modelos de Banco de Dados e Arquiteturas de Dados
*   **`AC4/`** – Deploy de Aplicações Django na AWS

---

## 🗂️ Organização Interna das Atividades

Cada pasta de atividade segue um padrão rigoroso de organização técnica e acadêmica:

*   **`README.md`:** Descrição detalhada, ementa e objetivos de aprendizagem específicos daquela atividade.
*   **`formulario.md`:** Questões originais propostas pelo corpo docente.
*   **`objetivas.md`:** Gabarito detalhado com as respostas de múltipla escolha *(Adicionado a partir da reestruturação da AC4)*.
*   **`discursivas.md`:** Resoluções teóricas, códigos técnicos e engenharia de arquitetura *(Adicionado a partir da reestruturação da AC4)*.
*   **`respostas.md`:** Arquivo mantido para manter o histórico de documentação e explicar o direcionamento para os novos arquivos revisados de respostas (`objetivas.md` e `discursivas.md`).

---

## 📌 Observações Técnicas e Governança

Este repositório adota e segue estritamente as boas práticas de organização acadêmica, versionamento com Git e documentação técnica profissional. Há uma separação clara entre os enunciados de avaliação e os blocos de engenharia de soluções, aplicando na prática conceitos avançados de Cloud Computing com foco no ecossistema **Amazon Web Services (AWS)**.
